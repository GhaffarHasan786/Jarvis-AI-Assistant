# Jarvis-AI-Assistant
"A voice-activated AI personal assistant build with Python that can perform tasks like web searching, launching app,and voice interaction."
### Features
**Voice Recognition:** Understand Human speech using Google speech Recoginition API
**Smart Serach:**  Can search for anything on google, youtube, and other apps
**System Automation:** Opens applications like VS code,Notepad,or Browser via voice.
**Real-time Info:** Tells the current time, date,and weather updates.
**Interactive Voice:** Respond back with a professional AI voice assistant.

### Tech Stack
**Language:** Python 3.x
**Libraries:** 'Speech_recognition', 'Pyttsx3', 'Wikipedia','Webbrowser'.
